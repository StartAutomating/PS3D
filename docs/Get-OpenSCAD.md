
Get-OpenSCAD
------------
### Synopsis
Gets OpenSCAD designs

---
### Description

Gets information about OpenSCAD designs.

---

### Examples
#### EXAMPLE 1
```PowerShell
Get-OpenSCAD -ScadFilePath .\MyDesign.scad
```
